- 1,770 simulated samples across three altitudes for the volumetric reconstruction.

## Isolation and provenance

The custom-engine workflow uses a unique per-run HoloOcean UUID plus private work, output, temp, cache, octree and log directories. The runner records process state and terminates only the exact engine process it launched. It never searches for or kills other HoloOcean workflows. Full implementation note: `../../brinewatch/docs/application/pfh2026/ISOLATED_EXECUTION.md`.
